# Tests
* **Sign-Up Page & Log-in Page** 
    * Martian will receive an error message if:
        1. Email does not follow standard *email@domain.com* format.
        2. User account already exists. 
        3. Any field is left blank

* **Profile/Home Page**
    * Martian will receive an error if attempting to save a task without a title. 

* **Task Page**
    * Martian will receive an error if attempting to save a new task without a title. 
    * The tasks are specific to the logged-in Martian.
